This is my personal portfolio website designed and developed to showcase my skills, projects, and experience as a Frontend Developer.

The website is fully responsive and built using HTML, CSS, and JavaScript. It includes smooth animations, interactive sections, a projects page linked with GitHub repositories, and a contact form for easy communication.

### Features
- Responsive design for all screen sizes  
- Animated sections using modern UI effects  
- Projects section with GitHub repository links  
- Skills, About, and Contact pages  
- Clean and professional UI  

### Technologies Used
- HTML5  
- CSS3  
- JavaScript  
- Font Awesome  
- bootstrap
- AOS
This portfolio represents my journey in frontend development and my passion for building user-friendly web experiences.
